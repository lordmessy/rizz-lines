Simon Game

Press any keyword to start game

Game Rules

  • Start the Game: Press any key on your keyboard to begin the game.
  • Watch the Sequence: The game will show you a sequence of colors. Pay close attention and remember the order!
  • Repeat the Sequence: Click on the buttons to repeat the same sequence of colors in the exact order shown.
  • Level Up: If you get the sequence right, you move to the next level, and the game adds one more color to the sequence.
  • Game Over: If you press the wrong color, the game ends. To play again, press any key to restart.
Example:
  • Level 1: The game flashes Yellow. You click the Yellow button. ✅
  • Level 2: The game flashes Yellow → Orange. You click Yellow, then Orange. ✅
  • Level 3: The game flashes Yellow → Orange → Grey. You click Yellow, Orange, and Grey. ✅
  • Level 4: The game flashes Yellow → Orange → Grey → Brown. You click Yellow, Orange, Grey, but then click the wrong button. ❌

Result: Game Over! The game resets, and you can start again by pressing any key.
